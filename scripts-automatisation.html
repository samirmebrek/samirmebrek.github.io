SCRIPTS D'AUTOMATISATION


Durant mon alternance, j'ai conçu un script d'automatisation pour optimiser les traitements récurrents sur Excel, qui impliquaient plus de 10 feuilles et étaient effectués chaque semaine ou sur demande. Ce projet a considérablement réduit le temps de travail de la personne en charge de ces tâches.

Le script que j'ai développé permet de se connecter à diverses bases de données, de recueillir des données de plusieurs applications et de les combiner avec la base de données principale de l'entreprise. Cela permet de calculer la performance des techniciens intervenant sur les ascenseurs. En exécutant le script, les résultats sont obtenus en moins de 3 secondes, et il est également possible d'envoyer automatiquement par e-mail certaines informations pertinentes.

L'avantage de ce script est son exécution rapide et la facilité avec laquelle les informations peuvent être distribuées, ce qui libère du temps pour des tâches plus stratégiques.

Ce projet ouvrait également la possibilité d'automatiser entièrement le processus à l'aide d'Azure Functions, ce qui éliminerait le besoin d'une exécution manuelle par l'utilisateur.

Technologies utilisées : SQL, Python, gestion des e-mails, pandas, gestion des erreurs, Azure functions, (excel, odbc, Time, smtplib) libraries ...
